open_your_heart
# こころの解放＆開放
「答えを出しても出さなくてもいいよ」

## アプリについて
このアプリは、日々の不安やモヤモヤをそっと吐き出すためのメンタルケア・チャットアプリです。
プロのカウンセラーのように、あなたの気持ちを否定せず、ただ優しく受け止めることを大切にしています。

## 特徴
- **否定しない共感**: AIがあなたの言葉を優しく受け止めます。
- **安心のプライバシー**: 入力した内容はあなたのブラウザにのみ保存され、サーバーには送信されません。
- **いつでもリセット**: 記録はいつでも自分で消去できます。
